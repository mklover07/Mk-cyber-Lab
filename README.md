## 📖 About

**MK Cyber Hub** is a next-generation cyber intelligence platform designed for **security researchers**, **OSINT investigators**, **journalists**, and **cybersecurity enthusiasts**. It combines real-time threat monitoring, OSINT tools, AI-powered analysis, and live FIFA World Cup updates in one modern, glassmorphism-styled interface.

> 🚀 **Empowering People Through Truth, Technology & Awareness**

---

## ✨ Features

### 🛡️ **Cyber Security & OSINT**
- 🔍 **10+ OSINT Tools** — Google Dorking, Shodan, Censys, TheHarvester, Spiderfoot, Maltego, WHOIS, VirusTotal, Have I Been Pwned, Wayback Machine
- 🤖 **AI Threat Analyzer** — AI-powered threat analysis for IPs and domains
- 🕵️ **Dark Web Monitor** — Check if your email appears in data breaches
- 🛡️ **Security Scanner** — Scan websites for SSL, headers, and open ports

### 🌍 **Live Intelligence**
- 🌍 **3D Interactive Globe** — Real-time globe with Three.js
- 📊 **Live Threat Map** — Simulated cyber attack visualization
- 📰 **Cyber News Feed** — Latest cybersecurity news updates
- 📊 **Threat Analytics** — Interactive charts (Chart.js)

### ⚽ **FIFA World Cup 2026**
- 🏆 **Live Stats** — Matches played, goals, top scorers, next matches
- 🔴 **Live Match Updates** — Real-time score simulation
- 📰 **FIFA News** — Latest football news updates

### 🎮 **Interactive Features**
- 🎮 **CTF Challenges** — Web Exploitation, Crypto, OSINT challenges
- 🎤 **Voice Commands** — Control tools with your voice (Chrome/Edge)
- 🌙 **Dark/Light Theme** — Toggle between dark and light modes
- 📱 **Mobile-First Design** — Fully responsive across all devices
